Setup — iPhone / iPad

  1. Settings → Calendar → Accounts → Add Account → Other → Add Subscribed Calendar.
  2. Long-press the link above on your phone → Copy. Paste into the Server field.
  3. Tap Next → Save. Repeat for the other two URLs.
  4. In the Calendar app, tick the new calendars under "Subscribed".

Setup — Mac

  1. Calendar app → File → New Calendar Subscription.
  2. Paste a URL → Subscribe. Set Auto-refresh to "Every hour".
  3. Repeat for each URL.

These are read-only subscribed calendars. To add or change events, text Oliver — he updates the rules and the next refresh picks it up.
